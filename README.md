- how to edit

responses can be edited by just changing whats in the .txt files

keybinds can be edited in configs ('enteringchat' / 'exitchat') variable

editing the time for each action can be done by changing the value within the 'spacetime' variable

typespeed is just smth so its not too fast and prevents the words turning to keyboard smash

if you cant find yourself tabbing in and out of the program quickly enough you can edit the 'waittime' variable that plays like a timer from when you hit your set trigger and when the function executes.
